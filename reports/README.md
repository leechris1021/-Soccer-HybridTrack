## reports

这个目录存放实验报告、灵敏性分析报告和结果总结文档。

### 当前文件

- `实验报告.md`
- `best_adaptive_tri_report.md`
- `sensitivity_report.md`
- `gate_weight_sensitivity_report.md`
- `rule_weight_sensitivity_report.md`
- `shape_weight_sensitivity_report.md`
- `原版.txt`

### 建议用途

- 把最终论文/答辩会用到的实验说明集中放这里
- 把自动生成的 Markdown 报告也统一收敛到这里
- 方便 GitHub 读者快速看到实验结论，而不是去翻输出目录

### 注意

如果某份报告是由脚本自动生成的，也建议在文内注明它来自哪个脚本和哪组参数。
